# Ramsey Theory DRP (University of Michigan)

This repository contains my Directed Reading Program (DRP) project focused on Ramsey theory and extremal graph constructions.

## Overview
- Studied Ramsey numbers and their asymptotic behavior
- Analyzed extremal graph constructions including Paley graphs
- Developed formal proofs of combinatorial bounds

## Key Topics
- Ramsey theory
- Extremal graph theory
- Probabilistic methods
- Paley graphs

## Outcome
- Co-authored a research-style paper under faculty mentorship
- Presented results through the University of Michigan DRP program
